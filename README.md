# HGEN670_GWAS_tutorial_2026

GWAS tutorial based on sgkit

## Notebooks

### Toy GWAS Tutorial (`toy_gwas_colab.ipynb`)

A beginner-friendly tutorial on Genome-Wide Association Studies (GWAS) that demonstrates:
- How to simulate genetic data with population structure
- Why population structure causes confounding in GWAS
- How Principal Components (PCs) help control for confounding
- How to interpret Manhattan plots and QQ plots
- The difference between unadjusted and PC-adjusted GWAS results
